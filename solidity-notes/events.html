Events

Use case

Write functions can't return data. We can use events instead. So events allows to return values, also store data

syntax declaration

event tokenReceived(address _from, address _to, uint _amount); allows up to 3 arguments than can be accesible by index

emitting the event

emit tokenReceived(msg.sender, receiverAddress, msg.value); This code will be inserted in the function we want.
