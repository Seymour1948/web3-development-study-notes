Function Modifiers

Public

Make the function callable from outside or inside

Private

Only callable from inside the contract, nor even derived contracts

External

Can be called from other contracts

Can be called from outside

Internal

Only the contract or derived contact can call this functions

View

Read the state but don't interact with it

Pure

Didn't interact with the state, for example and amount convert wei to eth

Writing vs Reading Functions

Writing functions require a transaction

Reading functions require a call

Call only require a local node so they are virtually free
