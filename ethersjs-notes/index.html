Ethers JS Notes

installation

npm install --save ethers in a empty directory after "npm init -y"

require

const { ethers } = require("ethers"); import library

provider

const provider = new ethers.providers.JsonRpcProvider("HTTP://0.0.0.0:7545") string argument for ganache

get address

const signer = await provider.getSigner(); obtain first account

transfer

signer.transfer({to: "0xdsds...", value: ethers.utils.parseEther("1.0")}); send ether from signer account

call

provider.call(from: signer, to: "0x989...", data: ethers.utils.kek256(ethers.utils.toUtf8Bytes("myUint()"))) Low level call

ethers.utils.kek256(ethers.utils.toUtf8Bytes("myUint()")) This part hash our string

branch

returns the current branch we are git branch

rename the branch to the given name git branch -m main

merge

being in main merge branch in it git merge myBranch

if a conflict arise you need to resolve and the commit the solved

delete files

You can just simply remove file and add this change or use git rm file.txt You could need to navigate to the folder you want to remove or specify the route the file is located. Only works with files previously added to the repository.

delete a commit

git reset --hard HEAD~1 step back one commit, deleting it.

delete a branch

git branch -D myBranch remove the specified branch. You should be in another branch to do this operation.

reset unstaged changes

git checkout -- . remove the unstaged changes. You can specify a given file to affect only this one.

unstaging staged files

git reset file.txt remove the file from staged ones. You can reset changes later on if you want the changes with the previous command.
