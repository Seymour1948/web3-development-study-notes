Hardhat Notes

installation

npm install --save-dev hardhat in an empty directory

run hardhat

npx hardhat in your project folder

install with dependencies

npm install --save-dev @nomiclabs/hardhat-waffle ethereum-waffle chai @nomiclabs/hardhat-ethers ethers this will install hardhat with dependencies

task

npx hardhat accounts this will run built in task accounts

compile contracts

npx hardhat compile send ether from signer account

run tests

npx hardhat test /tests/your-test.js Run your tests included in your tests folder

deploy your contracts

npx hardhat run scripts/sample-script.js Use the included boiler plate to deploy your contracts

use --network to identify in which network would be deployed. It should be already configured in hardhat-config.js file
